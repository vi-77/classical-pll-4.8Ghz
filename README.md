# classical-pll-4.8Ghz
Design and transient simulation of a 4.8 GHz Classical Charge-Pump Phase-Locked Loop (PLL) in Cadence Virtuoso.
This repository contains the design, schematic capture, and transient simulation verification of a classical Charge-Pump Phase-Locked Loop (PLL) targeting a lock frequency of 4.8GHz. The circuit was simulated in Cadence Virtuoso using the GPDK 45nm process kit to verify dynamic loop locking behavior, control voltage settling, and output frequency accuracy.
